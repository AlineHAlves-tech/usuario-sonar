# Usuarios Sonar

Projeto Java simples para testar o SonarQube Cloud.

## Métricas analisadas

Este projeto permite visualizar no SonarQube Cloud:

- Bugs
- Vulnerabilidades
- Code Smells
- Cobertura de Testes
- Duplicação de Código
- Maintainability
- Reliability
- Security
- Quality Gates
- Quality Profiles

## Sobre o sistema

O sistema simula um cadastro básico de usuários com:

- Cadastro
- Listagem
- Busca por ID
- Busca por email
- Autenticação simples
- Desativação de usuário
- Geração de token
- Relatório de usuários

## Observação

O projeto tem alguns pontos de código propositalmente simples para que o SonarQube Cloud consiga analisar e exibir sugestões de melhoria.
